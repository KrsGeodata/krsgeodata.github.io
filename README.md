# Kommunal forskning, innovasjon og eksperimentering i praksis

Dette nettstedet fungerer som inngangsporten til Kristiansand kommunes åpne arbeid innen forskning, utvikling og innovasjon (FoU).

Gjennom samarbeid mellom kommune, akademia, studenter, offentlige virksomheter og næringsliv utforsker vi nye arbeidsformer, teknologier og tjenestekonsepter før større investeringer og anskaffelser gjennomføres.

Målet er enkelt:

> Forstå problemet først. Spesifiser deretter. Anskaff til slutt.

---

## Om FoU-programmet

FoU-programmet er etablert for å styrke kommunens evne til å:

- Utforske reelle problemstillinger
- Validere behov og hypoteser
- Utvikle og teste MVP-er
- Bygge intern kompetanse
- Redusere risiko før anskaffelser
- Dele erfaringer åpent

Arbeidet omfatter blant annet:

- Kunstig intelligens
- Plan- og byggesaksområdet
- Geodata og geografiske informasjonssystemer
- Digitale innbyggertjenester
- Datadeling og samhandling
- Kommunal digitalisering

## Arbeidsprinsipper

Vi tror på:

- Åpen innovasjon
- Praktisk eksperimentering
- Tverrfaglig samarbeid
- Tidlig validering
- Kunnskapsdeling
- Gjenbruk på tvers av sektoren

Når det er mulig publiseres dokumentasjon, erfaringer, metoder og kode åpent slik at andre kommuner og virksomheter kan bygge videre på resultatene.
